------

**核心机制**：修改分两层防线——

1. **静态映射表** `fixed_type_map.json`：直接将 `HashMap`、`HashSet` 等裸类型映射为带 `AnyHashable` 的泛型形式，确保简单查找路径不遗漏
2. **动态替换逻辑** `get_cangjie_type()`：处理运行时递归解析泛型的场景（如 `HashMap<Object, String>` → 递归把 Object 解析为 Any → 检测到 key 位置是 Any → 替换为 AnyHashable），确保带显式泛型参数的输入也不会漏掉

### 其他修改

#### 1. `Dockerfile`

在 Dockerfile 里用 sdkman 安装了Java，因为 sdkman 的环境变量只在交互式shell 中生效，而 RUN 指令启动的是非交互式 shell，没有自动 source sdkman-init.sh，所以JAVA HOME 和 PATH 都没被设置。

* 第72行,修改为:

  ```
  RUN source "$HOME/.sdkman/bin/sdkman-init.sh" && mvn clean install -DskipTests
  ```

  在文件最后添加

  ```

    - pip:
        - rank_bm25>=0.2.0
  ```
